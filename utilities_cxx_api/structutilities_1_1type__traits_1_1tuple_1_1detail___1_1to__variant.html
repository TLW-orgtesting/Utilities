struct utilities::type_traits::tuple::detail_::to_variant< T >

Primary template for converting a std::tuple to a std::variant.

This is the primary template for converting a std::tuple to a std::variant. This template is not defined leading to a compiler error when T is not an instantiation of std::tuple.

Template Parameters
TThe type we are attempting to convert to an std::variant. For the primary template T will be any type aside from std::tuple and the code will fail to compile.
