struct utilities::type_traits::tuple::detail_::to_variant< std::tuple< Args... > >

Specialization of tuple_to_variant for when the template paramter is a std::tuple.

This specialization of tuple_to_variant is for when the template type parameter is an instantiation of std::tuple. This class will contain a member type type set to the type of a std::variant with the same template parameters as the std::tuple it is specialized on.
