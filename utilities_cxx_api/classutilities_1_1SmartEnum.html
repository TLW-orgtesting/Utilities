Detailed Description

template<class T>
class utilities::SmartEnum< T >

A class to implement smart enumerations, based on strings.

I stole the basic design of this class from here. That said the actual implementation is a bit different in that I am not concerned with being able to iterate over all the possibilities. This means I can dodge the weird static std::set instance. It also means I can get the entire class to be a compile-time literal.
