template<typename Type2Look4, typename Tuple>
struct utilities::type_traits::tuple::detail_::has_type< Type2Look4, Tuple >

Primary template for determining if a type is in a tuple.

This struct can be used to determine if Type2Look4 appears in the tuple Tuple. The primary template is selected when Tuple is not an std::tuple; the primary template is not defined and thus a compilation error will arise if Tuple is not a tuple.

Template Parameters
Type2Look4The type we are attempting to find in Tuple
TupleIf this version of has_type is selected Tuple will be a type other than std::tuple.
