Detailed Description

Class used to demangle RTTI names.

In C++ you can get RTTI of a type using the built-in typeid function. You can then use the resulting class for comparisons. At first the name member of the resulting class seems particularly useful for printing type information; however, one quickly realizes that the result is the platform- specific mangled type name. The Demangler class wraps the process of demangling the name thereby providing much nicer print outs of types.

At the moment this class only knows how to demangle the GNU compilers.

std::string utilities::printing::Demangler::demangle ( const char *  t)

Demangles the type inside t.

The name of the type stored inside t is compiler-specific. This function will call the compiler intrinsic function to demangle it and return the result. The implementation for the GNU compiler is shamelessly stolen from https://stackoverflow.com/a/4541470.

Note
This overload is the one that does all of the work. The other two overloads simply defer to this one.
