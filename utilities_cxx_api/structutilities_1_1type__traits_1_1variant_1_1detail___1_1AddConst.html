template<typename T>
struct utilities::type_traits::variant::detail_::AddConst< T >

Primary template for adding const to all types in a variant.

The primary template is not defined and results in a compilation error if T is not a specialization of std::variant.

Template Parameters
TThe type of the variant we are adding const to. If this template is selected it means T is not actually a specialization of std::variant.
