template<typename T>
struct utilities::is_iterator< T >

This struct will contain a value true if the type satisfies the C++ concept of "Iterator".

Warning
This type will not check if the type is swappable. A type_trait, is_swappable exists in C++17 for this purpose. Implementing our own is a royal pain on account of how the STL actually does swapping...

The full specificiation of what an "iterator" must have is available < a href="http://en.cppreference.com/w/cpp/concept/iterator">here.
